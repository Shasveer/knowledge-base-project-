# Knowledge Base Navigation Index

## Learning Roadmap

1. **Start with Git Basics** (knowledge-base/git/)
   - Begin with [git-init.md](knowledge-base/git/git-init.md) to set up repositories
   - Learn branching in [git-branch.md](knowledge-base/git/git-branch.md) for parallel work
   - Master commits with [git-commit.md](knowledge-base/git/git-commit.md)
   - Understand merging in [git-merge.md](knowledge-base/git/git-merge.md)
   - Sync with remotes using [git-pull-push.md](knowledge-base/git/git-pull-push.md)
   - Save work temporarily with [git-stash.md](knowledge-base/git/git-stash.md)

2. **Master Markdown Syntax** (knowledge-base/markdown/)
   - Structure documents with [headings.md](knowledge-base/markdown/headings.md)
   - Organize content using [lists.md](knowledge-base/markdown/lists.md)
   - Link to resources with [links.md](knowledge-base/markdown/links.md)
   - Display code in [code-blocks.md](knowledge-base/markdown/code-blocks.md)
   - Format data with [tables.md](knowledge-base/markdown/tables.md)

3. **Apply Collaborative Workflows** (knowledge-base/workflows/)
   - Use branches effectively in [branching-workflow.md](knowledge-base/workflows/branching-workflow.md)
   - Review changes with [pr-reviews.md](knowledge-base/workflows/pr-reviews.md)
   - Track tasks via [issue-tracking.md](knowledge-base/workflows/issue-tracking.md)
   - Collaborate globally in [collaboration.md](knowledge-base/workflows/collaboration.md)
