# Lists

## Definition

Lists in Markdown use `-` or `*` for unordered, numbers for ordered.

## Explanation

Lists organize information in readable formats.

## Example

- Item 1
- Item 2

1. First
2. Second

## Command block

```markdown
- Unordered item
1. Ordered item
```

## Real-world use case

README files use lists to outline features or steps.
