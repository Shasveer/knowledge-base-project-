# Tables

## Definition

Tables in Markdown use pipes `|` and dashes `-`.

## Explanation

Tables organize data in rows and columns.

## Example

| Column 1 | Column 2 |
|----------|----------|
| Data 1   | Data 2   |

## Command block

```markdown
| Header | Header |
|--------|--------|
| Cell   | Cell   |
```

## Real-world use case

Documentation uses tables for comparisons or data summaries.
