# Markdown Syntax

## Concept

Markdown is a lightweight markup language for formatting text.

## Explanation

Markdown uses simple syntax to create formatted documents.

## Example

Use # for headings, - for lists, **bold** for emphasis.

## Command

```markdown
# Heading 1
## Heading 2

- List item 1
- List item 2

[Link text](url)
```

## Use case

Documentation, README files, and notes use Markdown for readability.
