# Headings

## Definition

Headings in Markdown use `#` symbols to create section titles.

## Explanation

Headings structure documents with different levels of importance.

## Example

# Main Title

## Subsection

### Sub-subsection

## Command block

```markdown
# Heading 1
## Heading 2
### Heading 3
```

## Real-world use case

Documentation uses headings to organize content for easy navigation.
