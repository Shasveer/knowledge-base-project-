# Formatting Guide

## Concept

Advanced Markdown formatting for code and tables.

## Explanation

Markdown supports code blocks and tables for structured content.

## Example

Use backticks for inline code, triple backticks for blocks.

## Command

```markdown
Inline `code` here.

```
Code block
```

| Header | Header |
|--------|--------|
| Cell   | Cell   |
```

## Use case

Technical documentation uses code blocks and tables for clarity.
