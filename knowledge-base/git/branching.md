# Branching

## Concept

Git branching allows parallel development streams.

## Explanation

Branches isolate changes, enabling safe experimentation and collaboration.

## Example

Create a feature branch, develop, then merge back.

## Command

```bash
git branch feature-x
git checkout feature-x
# work on feature
git checkout main
git merge feature-x
```
