# Git Basics

## Concept

Git basics cover fundamental operations for version control.

## Explanation

Git tracks changes to files and coordinates work among multiple people.

## Example

Initialize a repo, make changes, commit, and push to remote.

## Command

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin <url>
git push -u origin main
```

## Use case

Developers use Git basics to start any new project with version control.
