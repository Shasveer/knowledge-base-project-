# Git Stash

## Concept

Git stash temporarily saves uncommitted changes without committing them.

## Explanation

Stashing allows you to switch branches or contexts without losing work, then restore later.

## Example

Stash changes before switching to another branch for urgent fixes.

## Command

```bash
git stash push -m "Work in progress"
git stash list
git stash pop
```

## Use case

Developers stash feature work to quickly address a bug fix on the main branch.
