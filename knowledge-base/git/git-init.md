# Git Init

## Definition

`git init` initializes a new Git repository in the current directory.

## Explanation

This command creates a `.git` folder that tracks changes to files in the project.

## Example

Run `git init` in an empty folder to start version control.

## Command block

```bash
mkdir my-project
cd my-project
git init
```

## Real-world use case

Developers use `git init` when starting a new project to enable version control from the beginning.
